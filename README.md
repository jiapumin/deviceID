##项目介绍
    该项目是开发者猩球计划中国电信第二期开源软件人才能力提升实战训练营项目之一，题目是来自于翼支付公司实际生产需求。

##项目成员


##项目目标
建立一套基于手机设备（首期目标是Android）的设备ID识别系统，并基于识别的结果，为业务情况提供风险程度的提示或告警，并通过直观的图形好界面，实现业务风险程度的实时展示

##项目进度
经过5天的紧张开发，项目已完成初期目标，并实现了全流程的功能验证。
